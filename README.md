
# THISDOT-CLI

## This is a simple CLI for checking Tax and Totals

## Installation

```bash
git clone https://github.com/nerdeveloper/thisdot-cli.git
cd  thisdot-cli
npm install

```

## Usage

```bash
npm link
thisdot-cli
```

### Examples

```bash
thisdot-cli --zipcode 20500 --subtotal 100000.00
```

## Lint

```bash
npm run lint
```

## Uninstall Package

```bash
npm unlink
```
